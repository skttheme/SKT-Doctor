== Description ==

SKT Doctor is a gutenberg block based theme which can be used for hospital, clinic, healthcare, pharmacy, medical store, science, lab, drug, medication, veterinary, maternity, healer, general physician, therapist, orthopaedic, eye clinic, pathology, dermatologist, gynecologist, ambulance, physiotherapy, health consultant, pediatrician, psychologist, dentist, nurse, weight loss and health care centre. Compatible with WooCommerce, contact form and SEO plugins.
